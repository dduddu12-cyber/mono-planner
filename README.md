# Mono Planner

개인용 주간/일간 To-Do 다이어리 웹앱  
버전: 1.0.0 | 저장 방식: 브라우저 IndexedDB + JSON 백업

---

## 실행 방법

### 방법 1: 로컬 파일 직접 열기

```
index.html 파일을 브라우저에서 직접 열기
```

- 파일 탐색기에서 `index.html`을 더블클릭하거나
- 브라우저 주소창에 파일 경로를 입력 (`file:///D:/...`)
- **서비스 워커(PWA)** 기능은 서버 환경에서만 동작합니다

> ⚠️ Safari에서 로컬 파일 열기 시 IndexedDB가 제한될 수 있습니다.  
> iPad에서는 서버 환경(아래 방법 2)을 권장합니다.

### 방법 2: VS Code Live Server 사용 (권장)

1. VS Code에 **Live Server** 확장 설치
2. `index.html`에서 우클릭 → **Open with Live Server**
3. 브라우저에서 `http://127.0.0.1:5500/mono-planner/index.html` 접속

### 방법 3: Python 간이 서버

```bash
# mono-planner 폴더에서 실행
cd "D:\■장선미 개인파일\01_클로드코드\mono-planner"
python -m http.server 8080
```

브라우저에서 `http://localhost:8080` 접속

### 방법 4: Node.js serve

```bash
npm install -g serve
serve "D:\■장선미 개인파일\01_클로드코드\mono-planner"
```

## iPad에서 홈 화면에 추가 (PWA)

1. iPad Safari에서 앱 주소 접속
2. 주소창 옆 **공유 버튼** (박스 위 화살표) 탭
3. **홈 화면에 추가** 선택
4. 이름 확인 후 **추가** 탭
5. 홈 화면에서 앱처럼 실행 가능

> PWA 아이콘 추가: `icons/` 폴더를 만들고  
> `icon-192.png` (192×192), `icon-512.png` (512×512) 파일을 넣으세요.

## 데이터 저장 방식

### 평소 작업

- 모든 데이터는 **브라우저 IndexedDB**에 자동 저장됩니다
- 앱을 닫아도 데이터가 보존됩니다
- 인터넷 연결 없이 사용 가능합니다

### 중요 규칙

- 브라우저 **캐시/사이트 데이터 삭제** 시 데이터가 지워집니다
- **시크릿 모드** 에서는 데이터가 저장되지 않습니다
- 정기적으로 **백업(JSON 내보내기)** 하는 것을 강권합니다

---

## 백업 방법

### JSON 파일로 백업

1. 상단 **[백업]** 버튼 클릭
2. 백업 요약 확인 (포함 주차 수, 전체 업무 수, 최근 수정일)
3. **[JSON 파일 다운로드]** 버튼 클릭
4. `planner-backup-YYYY-MM-DD.json` 파일 저장됨
5. 다운로드된 파일을 **OneDrive** 등 원하는 위치로 이동

### 권장 백업 주기

- 매주 금요일 백업 권장
- 중요한 데이터 입력 후 즉시 백업
- 백업 파일명에 날짜가 포함되므로 여러 버전 보관 가능

---

## 복원 방법

### 다른 기기에서 데이터 불러오기

1. 백업 파일(`.json`)을 해당 기기로 이동
2. 앱에서 상단 **[복원]** 버튼 클릭
3. **파일 선택**으로 백업 파일 선택
4. 미리보기 확인 (포함 주차 수, 업무 수, 내보낸 날짜)
5. 불러오기 방식 선택:
   - **덮어쓰기**: 현재 데이터를 모두 지우고 백업으로 교체
   - **병합하기**: 현재 데이터에 백업 데이터를 합침 (중복 제외)
6. **[불러오기 실행]** 클릭

## 주요 기능

| 기능 | 설명 |
|------|------|
| 주간 To-Do | 이번 주 핵심 업무 관리 (우선순위/태그/메모) |
| 일간 To-Do | 월~일 7일 구조로 날짜별 업무 관리 |
| 체크박스 + 취소선 | 완료 표시 시 자동 취소선 |
| 주간/일간 메모 | 자동 저장되는 텍스트 메모 |
| 필기 캔버스 | Apple Pencil/터치/마우스 필기 지원 |
| 이월 기능 | 미완료 업무를 다음 주로 이월 |
| 히스토리 | 지난 주 업무/메모/필기 열람 |
| 휴지통 | 삭제된 항목 복원 (최근 50개) |
| 검색 | 전체 업무 텍스트 검색 |
| 필터 | 상태/우선순위/태그별 필터 |
| 다크모드 | 라이트/다크 테마 전환 |
| PWA | iPad 홈 화면에 추가 지원 |

## 주의사항

- **Safari(iOS/iPadOS)**: 로컬 파일 실행보다 서버 환경을 권장합니다
- **브라우저 업데이트**: 드물게 IndexedDB 구조가 초기화될 수 있습니다. 정기 백업 필수
- **시크릿 모드 금지**: 데이터가 저장되지 않습니다
- **OneDrive 자동 동기화 없음**: 의도적인 설계입니다. 백업 파일을 직접 OneDrive로 이동하세요
- **아이콘 파일**: `icons/` 폴더와 PNG 파일을 직접 추가해야 PWA 아이콘이 표시됩니다
