# Mono Planner - 앱 요구사항 정의서

작성일: 2026-04-08  
버전: 1.0.0

---

## 1. 프로젝트 개요

### 목적
단순한 체크리스트가 아닌 "주간 계획 + 일간 실행 + 텍스트 메모 + 손필기 메모 + 기록 보존 + 이월 관리 + 상단 요약 위젯 카드"가 가능한 다이어리형 생산성 웹앱

### 배경
- Microsoft OneNote 개인계정/회사계정 충돌 문제 해결
- Microsoft 로그인 기반 동기화 없이 브라우저 로컬 저장 + JSON 백업/복원 방식 채택
- iPad와 노트북 모두에서 편리하게 사용

### 사용 환경
- 주 사용 디바이스: iPad, Windows 노트북
- iPad: Apple Pencil/터치 기반 필기
- 노트북: 키보드 타이핑 중심
- PWA 구조 포함 (iPad Safari 홈 화면 추가 지원)

---

## 2. 핵심 사용 패턴

사용자는 매주 다음 흐름으로 앱을 사용:
1. 이번 주에 해결해야 할 "주간 To-Do" 정리
2. 각 날짜별 세부 업무를 "일간 To-Do"로 분류
3. 완료한 업무는 체크 → 취소선 표시
4. 완료하지 못한 업무는 다음 주로 이월
5. 날짜별 메모와 손필기 메모 기록
6. 주기적으로 JSON 백업 → OneDrive에 보관

## 3. 저장 및 백업 방식

| 항목 | 방식 |
|------|------|
| 평소 저장 | 브라우저 IndexedDB (자동) |
| 백업 | 전체 데이터를 JSON 파일로 export |
| 백업 파일명 | `planner-backup-YYYY-MM-DD.json` |
| 백업 보관 | 사용자가 직접 OneDrive로 이동 |
| 복원 | 다른 기기에서 JSON 파일 import |
| Import 방식 | 덮어쓰기 / 병합하기 두 가지 제공 |
| 클라우드 동기화 | 없음 (의도적) |
| 자동 OneDrive 연동 | 없음 (의도적) |
| 구조 원칙 | 로컬 우선(Local-First) |

### 백업 파일 포함 정보
- `appVersion`: 앱 버전
- `dataVersion`: 데이터 구조 버전
- `exportedAt`: 내보낸 일시 (ISO 형식)
- `weeks`: 전체 주차 데이터
- `meta`: 마지막 백업/복원 시각

### 복원 미리보기 제공 항목
- 포함된 주차 수
- 전체 업무 수
- 최근 수정일
- appVersion

## 4. 데이터 구조

### 주차 키 형식
`YYYY-Wxx` (ISO 주차)  
예: `2026-W15`

### 주차 데이터 구조
```json
{
  "2026-W15": {
    "weekKey": "2026-W15",
    "weeklyTodos": [],
    "dailyTodos": {
      "2026-04-07": []
    },
    "weeklyNote": "",
    "weeklyNoteAt": null,
    "dailyNotes": {},
    "dailyNotesAt": {},
    "drawings": {},
    "createdAt": "ISO-string",
    "updatedAt": "ISO-string"
  }
}
```

### 업무(Todo) 항목 구조
```json
{
  "id": "고유 ID",
  "text": "업무 내용",
  "completed": false,
  "createdAt": "ISO-string",
  "updatedAt": "ISO-string",
  "completedAt": null,
  "priority": "high | medium | low",
  "tags": [],
  "memo": "",
  "linkedWeeklyTodoId": null,
  "carriedOverFrom": null,
  "carriedOverTo": null,
  "deletedAt": null
}
```

## 5. 디자인 컨셉

- **색상**: 무채색 기반 (흰색 / 검정 / 회색)
- **느낌**: 미니멀, 차분, 정돈된 다이어리 + 현대적 생산성 앱
- **특징**: 과하게 화려하지 않음, 깔끔한 레이아웃
- **터치**: iPad에서 손가락/펜으로 누르기 쉬운 크기
- **키보드**: 노트북에서 키보드 입력 효율 우선
- **테마**: 다크모드 / 라이트모드 지원
- **인쇄**: 인쇄 친화 스타일 포함

## 7. 반응형 레이아웃

### 데스크탑/노트북 (> 900px)
```
┌────────────────────────────────────────────────────────┐
│ [■ Mono Planner] [< W15 >] [오늘] [검색] [이월][백업][복원][◐] │
├────────────────────────────────────────────────────────┤
│ [오늘 할일] [진행률] [주간 미완료] [일간 미완료] [이월] [백업] [복원] │
├────────────────────────────────────────────────────────┤
│ [상태 필터] [우선순위 필터] [정렬] [태그 필터]          │
├──────────────┬───────────────────┬─────────────────────┤
│  주간 To-Do  │    일간 To-Do     │  날짜 상세           │
│  + 추가      │  월/화/수/목/금   │  (선택 날짜 메모)    │
│  [항목들]    │  토/일 카드 구조  │  + 드로잉 캔버스     │
│              │                   │                      │
│  주간 메모   │                   │                      │
├──────────────┴───────────────────┴─────────────────────┤
│  히스토리 패널 (접기/펼치기)                            │
│  휴지통 패널 (접기/펼치기)                              │
└────────────────────────────────────────────────────────┘
```

### iPad 세로 / 모바일 (≤ 900px)
- 하단 탭 네비게이션: 주간 | 일간 | 메모 | 히스토리
- 각 탭 전환으로 패널 표시
